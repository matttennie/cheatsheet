# vim

## Movement (Normal mode)
     h j k l                   Left down up right
     w b                       Forward/back word
     0 $                       Start/end of line
     gg G                      Top/bottom of file
     Ctrl+u Ctrl+d             Half page up/down
     Ctrl+f Ctrl+b             Full page forward/back

## Editing (Normal mode)
     i                         Insert before cursor
     a                         Append after cursor
     o O                       New line below/above
     x                         Delete char
     dd                        Delete line
     yy                        Yank line
     p P                       Paste after/before
     u                         Undo
     Ctrl+r                    Redo

## Visual mode
     v                         Character visual
     V                         Line visual
     Ctrl+v                    Block visual
     d y                       Delete/yank selection

## Command mode (:)
     :w                        Save
     :q                        Quit
     :wq or ZZ                 Save & quit
     :q!                       Quit without save
     :/pattern                 Search
