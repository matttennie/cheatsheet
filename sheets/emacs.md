# emacs

## Movement
     C-f C-b                   Forward/back char
     C-n C-p                   Next/previous line
     C-a C-e                   Start/end of line
     M-f M-b                   Forward/back word
     M-v C-v                   Page up/down
     M-< M->                   Beginning/end of buffer

## Editing
     C-d                       Delete forward char
     C-k                       Kill to end of line
     C-y                       Yank last killed
     C-_ or C-/                Undo
     C-x C-s                   Save buffer
     C-x C-c                   Exit

## Files & Buffers
     C-x C-f                   Open/find file
     C-x C-w                   Save as
     C-x b                     Switch buffer
     C-x k                     Kill buffer
