# top

## Navigation & View
     Space or Enter            Refresh
     q                         Quit
     h                         Help
     1                         Toggle per-CPU stats
     z                         Toggle color
     c                         Toggle full command line
     M P T                     Sort by mem/CPU/time

## Process Control
     k                         Kill (enter PID + signal)
     r                         Renice priority
     f                         Manage fields/columns
     u                         Filter by user

## Scrolling
     Arrow keys / PgUp PgDn   Scroll list
